# energy_consumption_analysis_system
An interactive Python-based dashboard for monitoring and predicting household power usage.

**Key Features
Real-time Metrics: Displays Avg. Usage (kW), Peak Hours, and Sub-metering stats.

Predictive Modeling: Uses Linear Regression to estimate energy demand based on Voltage.

Visual Analytics: Interactive Line, Bar, and Stacked Bar charts for trend analysis.

Smart Breakdown: A normalized Pie Chart showing appliance-specific consumption (AC, Laundry, General Use).

Dynamic Filtering: Filter data by time categories (Peak, Off-Peak, Mid-Day).

Tech Stack
Language: Python

Framework: Dash (Plotly)

Data/ML: Pandas, Scikit-learn (Linear Regression)

UI: Custom CSS / Dark Mode
